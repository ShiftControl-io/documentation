## Overview

Quickly and easily create and manage API keys for your organization and individual users, enabling you to authenticate and authorize API requests to ShiftControl, automate various functions, and integrate with other applications.

## Adding an API Key


<img
    src="/img/ShiftControl/API/new-api-key.png"
    alt="Add an API key"
/>


To add an API key, follow these steps:

1. **Navigate to API Key Management**: Click on the Cog at the top of the dashboard, go to the **API Keys** section under either the Org Settings or User Settings, depending on the type of API key you want to create.
2. **Click Add API Key**: Click the [Add API Key](/using-shiftcontrol/API/Adding-an-api-key) button located at the top of the screen.
3. **Fill in the Details**: In the form that appears, enter the details for the new API key, including its name, purpose, and expiry date.
4. **Save the API Key**: Click the **Save** button to create the API key.
5. **Copy the API Key**: After saving, copy the API key to a secure location as it will not be displayed again.

<img
    src="/img/ShiftControl/API/new-api-key-copy.png"
    alt="Copy your API key"
/>


<Admonition type="warning">
    Use Org API keys with caution as they have full administrative rights.
</Admonition>
