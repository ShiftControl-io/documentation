## Overview

The Settings area is organized into sections based on what you're configuring and who it affects:

### Personal Settings
Your individual account — profile details, theme preferences, and personal API keys. Changes here only affect you, not your organization.

- **[Account Settings](/using-shiftcontrol/Settings/account-settings/)** — Your name, email, and profile information
- **[Appearance](/using-shiftcontrol/Settings/appearance/)** — Light mode, dark mode, and display preferences
- **[User API Keys](/using-shiftcontrol/Settings/user-api-keys/)** — Personal API keys for automation and scripting

### Organization Settings
Settings that affect your entire organization. Only admins can modify these.

- **[Organization Details](/using-shiftcontrol/Settings/org-details/)** — Company name, domain, and features
- **[Departments](/using-shiftcontrol/Settings/departments/)** — Department definitions that drive dynamic group memberships
- **[Locations](/using-shiftcontrol/Settings/locations/)** — Office locations for location-based groups and policies
- **[Group Tags](/using-shiftcontrol/Settings/group-tags/)** — Color-coded categories for organizing groups
- **[User Roles](/using-shiftcontrol/Settings/user-roles/)** — ShiftControl admin role assignments
- **[Organization API Keys](/using-shiftcontrol/Settings/org-api-keys/)** — Org-level API keys for integrations

### Connectors
Integrate external systems with ShiftControl to sync identity data, automate provisioning, and unify management.

- **[Directories](/using-shiftcontrol/Settings/directories/)** — Connect identity providers (JumpCloud, Google Workspace) and HRIS systems
- **[App Connectors](/using-shiftcontrol/Settings/app-connectors/)** — Connect apps for usage data and cost tracking
